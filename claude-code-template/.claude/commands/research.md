description: Research a topic, synthesize findings, and produce a structured deliverable.
argument-hint: "<topic or question>"

1. **Scope.** Clarify the research question. If ambiguous, ask up to 3 clarifying questions before proceeding.
2. **Research.** Use available tools (web search, MCP sources, local files) to gather information. Prioritize:
   - Primary sources over secondary
   - Recent over dated
   - Peer-reviewed or official over blog posts
3. **Synthesize.** Write findings to a Markdown file in the working directory. Structure:

   ```
   # [Topic] — Research Summary
   **Date:** [today]
   **Question:** [the research question]

   ## Key Findings
   [Numbered findings, most important first]

   ## Analysis
   [Your synthesis — patterns, contradictions, gaps]

   ## Sources
   [Numbered list with URLs and access dates]

   ## Confidence & Caveats
   [What you're confident about, what's uncertain, what needs more investigation]
   ```

4. **Deliver.** Present the file. Flag any areas where the human should verify independently.
